# Go eCommerce Backend with Stripe Integration

A backend-only eCommerce system built with Go and the Gin framework, featuring product management, cart management , ordering and secure payment processing via Stripe, admin product management, cart handling, and JWT-based authentication.

---

## Features

- **Online Ordering System** – Customers can place orders and manage their carts.
- **Product Management** – Admins can add, update, and delete products.
- **Payment Processing** – Stripe PaymentIntents used for secure payment handling.
- **Webhook Handling** – Automatic processing of payment events and transaction updates.
- **JWT Authentication** – Secure user login and protected routes.
- **Transaction Handling** – Safe database operations with payment confirmation.
